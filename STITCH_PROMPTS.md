# Stitch 提示词模板

用于 Google Stitch 生成新页面的提示词模板集合

---

## 📱 交易页面 (Trade Page)

```
创建一个移动端加密货币交易页面，包含：

【布局】
- 顶部：返回按钮 + "交易" 标题
- 市场选择器：虚拟币/A 股/美股/港股 横向滚动标签
- 交易卡片：支付输入框 + 汇率显示 + 接收输入框
- 交易信息：汇率、手续费、预计到账
- 买入/卖出切换按钮
- 底部：确认交易按钮

【样式】
- 深色主题，背景 #0f0f0f
- 卡片背景 #1a1a1a，圆角 24px
- 输入框：深灰色背景 #2a2a2a，聚焦时青绿色边框
- 主按钮：青绿色渐变 #00d4aa → #6c5ce7
- 字体：Google Sans + Roboto Mono (数字)

【组件】
- 输入框：56px 高度，大字号 (20px)，显示占位符
- 市场标签：胶囊形状，激活状态渐变背景
- 汇率信息：小卡片，灰色背景，两行信息
- 交易按钮：全宽，48px 高度，带阴影和光晕

【交互】
- 输入框聚焦：边框变青绿色 + 发光效果
- 按钮按压：缩放 0.96 + 涟漪动画
- 市场切换：平滑滑动过渡
- 数字键盘：移动端自动弹出数字键盘

【无障碍】
- 输入框有清晰 label
- 按钮有足够对比度
- 触摸目标最小 48px
- 支持键盘导航

设备类型：MOBILE
```

## 📊 行情页面 (Market Page)

```
创建一个移动端市场行情页面，包含：

【布局】
- 顶部：页面标题 + 搜索图标
- 市场筛选：虚拟币/A 股/美股/港股 胶囊标签
- 涨跌筛选：全部/涨幅榜/跌幅榜 三段控制器
- 搜索框：带放大镜图标，圆角 12px
- 行情列表：每行显示币种/股票信息
- 底部：5 个标签导航栏

【列表项布局】
左侧：图标 + 名称 + 代码
中间：当前价格
右侧：涨跌幅（绿色上涨/红色下跌）

【样式】
- 深色主题
- 列表项：卡片式，背景 #1a1a1a，圆角 16px
- 间距：列表项之间 12px
- 涨跌幅标签：胶囊形，半透明背景

【数据展示】
- 价格：Roboto Mono 等宽字体，右对齐
- 涨跌幅：带背景色块，绿色#00c853/红色#ff5252
- 图标：圆形，40x40px，带品牌色背景

【交互】
- 列表项：点击高亮，右滑查看详情
- 标签切换：平滑过渡，激活状态渐变
- 下拉刷新：显示加载指示器
- 搜索：实时过滤列表

【动效】
- 页面切换：淡入 + 上滑
- 列表项：悬浮右移 4px
- 价格更新：闪烁提示（绿色/红色）

设备类型：MOBILE
```

## 💼 持仓页面 (Portfolio Page)

```
创建一个移动端持仓管理页面，包含：

【布局】
- 顶部：页面标题 + 刷新按钮
- 总资产卡片：渐变背景，显示总估值和 24h 盈亏
- 资产分布饼图：圆形图表，显示各币种占比
- 持仓列表：每个持仓显示币种、数量、价值、盈亏
- 空状态：无持仓时显示引导去交易

【资产卡片】
- 背景：青绿色渐变
- 总资产：42px 大字号，白色
- 涨跌幅：胶囊标签，半透明白色背景
- 光泽动画：缓慢移动的高光

【持仓列表项】
- 左侧：币种图标 + 名称 + 市场标签
- 右侧：持仓价值 + 24h 盈亏
- 详情展开：显示持仓数量、均价、成本

【样式】
- 卡片圆角：24px (资产卡), 16px (列表项)
- 字体：Google Sans + Roboto Mono
- 颜色：上涨绿色 #00c853，下跌红色 #ff5252

【空状态】
- 图标：64px，半透明
- 文字："暂无持仓"
- 按钮："去交易"，主色按钮

【交互】
- 列表项：点击展开详情
- 刷新：下拉刷新持仓数据
- 饼图：点击扇区高亮对应持仓

设备类型：MOBILE
```

## 📜 历史记录页面 (History Page)

```
创建一个移动端交易历史页面，包含：

【布局】
- 顶部：页面标题 + 筛选图标
- 时间筛选：全部/本周/本月/自定义 标签
- 交易类型筛选：全部/买入/卖出 分段控制器
- 历史列表：按时间倒序排列
- 空状态：无记录时显示提示

【列表项布局】
左侧：图标（买入绿色/卖出红色）+ 交易对 + 时间
右侧：交易数量 + 交易金额

【样式】
- 列表项：卡片式，背景 #1a1a1a，圆角 16px
- 图标：圆形 44x44px，买入绿色背景/卖出红色背景
- 时间：小字号，灰色 #a0a0a0
- 金额：Roboto Mono 字体

【分组】
- 按日期分组：今天/昨天/本周/更早
- 分组标题：小字号，灰色，sticky 定位

【空状态】
- 图标：📜 64px，半透明
- 文字："暂无交易记录"
- 说明：开始交易后记录将显示在这里

【交互】
- 列表项：点击查看详情
- 筛选：平滑过渡，实时更新列表
- 下拉刷新：更新历史记录

【详情弹窗】
- 交易类型图标
- 交易对和金额
- 时间戳
- 手续费
- 状态（已完成/处理中）

设备类型：MOBILE
```

## 🎯 通用组件提示词

### 底部导航栏
```
创建一个移动端底部导航栏组件，包含：

【布局】
- 5 个导航项：首页/交易/行情/持仓/历史
- 固定底部，毛玻璃背景
- 每个项：图标 + 文字标签

【样式】
- 背景：rgba(26, 26, 26, 0.95) + backdrop-filter blur(20px)
- 高度：80px (包含安全区域)
- 边框：顶部 1px 灰色边框
- 图标：26px，激活状态青绿色
- 文字：11px，激活状态加粗

【交互】
- 激活状态：图标下方显示 20px 青绿色指示条
- 按压反馈：缩放 0.95
- 图标动画：按压时缩放 0.9

【无障碍】
- 触摸目标：最小 48x48px
- 激活状态明显
- 支持键盘导航

设备类型：MOBILE
```

### 资产卡片
```
创建一个移动端资产展示卡片组件，包含：

【布局】
- 标签："总资产估值 (USD)"
- 金额：大字号显示
- 涨跌幅：胶囊标签

【样式】
- 背景：青绿色渐变 #00d4aa → #6c5ce7
- 圆角：32px
- 内边距：32px
- 金额：42px，Roboto Mono，白色
- 标签：13px，半透明白色
- 涨跌幅：半透明白色背景，胶囊形

【动效】
- 光泽动画：缓慢移动的高光
- 数字滚动：金额变化时滚动效果
- 闪烁：更新时短暂高亮

【无障碍】
- 高对比度文字
- 清晰的视觉层次

设备类型：MOBILE
```

## 💡 使用建议

1. **复制提示词** 到 Stitch 网页版
2. **调整细节** 根据你的具体需求
3. **生成变体** 点击 Variants 生成 3 个版本对比
4. **迭代优化** 基于生成结果调整提示词
5. **导出代码** 满意后导出 HTML/CSS
